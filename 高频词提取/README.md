# 高频词提取

基于jieba分词对情感分析结果进行词频统计，提取各版本×情感的高频关键词。

## 文件说明

| 文件 | 说明 |
|---|---|
| [jieba高频词提取.py](./jieba高频词提取.py) | 主脚本：加载自定义词库和停用词表，对评论按版本×情感分组分词，提取Top15高频词 |
| [jieba游戏词库.txt](./jieba游戏词库.txt) | 自定义jieba词库，包含崩坏星穹铁道专用术语（角色名、习语、游戏系统等），确保分词准确 |
| [停用词表.txt](./停用词表.txt) | 中文通用停用词表，过滤无意义高频词汇（的、了、是、我、你等） |
| [版本情感高频词.csv](./版本情感高频词.csv) | 输出结果：各版本×情感（正向/负向）的Top15高频词及频次 |

## 分词处理

1. 加载自定义游戏词库到jieba，确保专有名词不被错误切分
2. 过滤规则：
   - 长度 < 2 的单字
   - 纯数字/百分比/英文短码（≤3字符）
   - 停用词表中的词汇
3. 按版本×情感分组统计词频
4. 输出每个版本的Top15正向/负向高频词

## 输出数据表

| 输出 | 说明 |
|---|---|
| `版本情感高频词.csv` | 各版本×情感的高频词（正向/负向各Top15） |
| `全版本情感高频词.csv` | 合并全版本后的情感高频词 |
| `freq_words_version` | 入库的版本高频词表 |
| `freq_words_overall` | 入库的全版本高频词表 |

## 使用方式

```bash
python jieba高频词提取.py
```

## 自定义词库

如需添加新的游戏术语，编辑 `jieba游戏词库.txt`，每行一个词，格式为：

```
词汇 词频 词性
```

例如：
```
星穹铁道 200000 nz
开拓者 120000 n
```
